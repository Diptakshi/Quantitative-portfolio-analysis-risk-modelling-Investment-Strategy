# Quantitative-portfolio-analysis-risk-modelling-Investment-Strategy
Built a Python-based financial model to analyze equity performance, including return, volatility, and correlation analysis across multiple stocks.
Developed portfolio optimization using Sharpe ratio and efficient frontier to identify optimal asset allocation.
Implemented risk management techniques including drawdown analysis and volatility tracking, and evaluated trading strategies using moving averages.
Extended analysis with time-series forecasting (LSTM) to explore predictive modeling of stock prices.
